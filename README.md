# NibInk
This is a repository for an ecommerce site for selling inks and fountain pens, developed by four students of the University of Salerno. The project also has an admin panel for managing products, orders, users and shipping methods. We're inspired by some of the existing e-commerce platforms competitors. All the documentation is in Italian. We implemented the MVC model.
